# Settings Overview

Access settings from the gear icon in the sidebar, or press the Settings module.

## Settings Categories

### Account
Personal settings for your user account:
- **Profile** - Name, email, avatar
- **Preferences** - Theme, language, display options
- **Keybindings** - Customize keyboard shortcuts
- **Modules** - Show/hide sidebar modules
- **Delete Account** - Permanently remove your account

### Organization
Settings that affect your entire organization (admin access required):
- **Vaults** - Create and manage file vaults
- **Members & Teams** - Users, teams, roles, permissions
- **Company Profile** - Organization name and branding
- **Serialization** - Part numbering schemes
- **File Metadata** - Custom columns for file properties
- **RFQ Settings** - Request for quote configuration
- **Backups** - Backup and restore options
- **Recovery Codes** - Database recovery codes

### Integrations
Connect to external services:
- **Supabase** - Database connection status
- **SolidWorks** - CAD integration (Windows only)
- **Google Drive** - Cloud storage sync
- **Odoo ERP** - Enterprise resource planning
- **Slack** - Team notifications
- **Webhooks** - Custom integrations
- **REST API** - API access configuration

### System
Technical settings:
- **Performance** - Caching and optimization
- **Logs** - View application logs
- **Dev Tools** - Developer options
- **About** - Version info and credits
